# Dependencies

- bcryptjs
- body-parser
- cookie-session
- EJS
- Express
- Node.js

## Getting Started
- Install all dependencies (using 'npm install' command)
- Run the development web server with 'node express_server.js' command
- Enter the address given in the command line after running server
